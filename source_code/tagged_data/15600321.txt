 Change 

 <code>i[0] == r[0]
r == g[1][0]
r == g[1][1]
</code> 

 to (respectively) 

 <code>i[0] = r[0]
r = g[1][0]
r = g[1][1]
</code> 

 <code>==</code> is a logical comparison (it tests whether the two values are equal and returns <code>False</code> or <code>True</code>). <code>=</code> is the assignment operator.
