 If you want to list part of your attributes, override <code>__dict__</code>: 

 <code>def __dict__(self):
    d = {
    'attr_1' : self.attr_1,
    ...
    }
    return d

# Call __dict__
d = instance.__dict__()
</code> 

 This helps a lot if your <code>instance</code> get some large block data and you want to push <code>d</code> to Redis like message queue.
